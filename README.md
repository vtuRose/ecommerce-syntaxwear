# SyntaxWear - Tênis e Sneakers Online 👟

A **SyntaxWear** é uma interface de e-commerce moderna e responsiva focada na venda de calçados (tênis e sneakers). O projeto foi desenvolvido com uma arquitetura de CSS modularizada, focando em boas práticas de organização e design.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias fundamentais da web:

- **HTML5:** Estrutura semântica para melhor acessibilidade e SEO.
- **CSS3:** Estilização avançada com o uso de:
  - **CSS Variables:** Gerenciamento centralizado de fontes e cores.
  - **CSS Flexbox/Grid:** Layouts flexíveis e grids complexos para exibição de produtos.
  - **Componentização:** Estilos separados por componentes (Header, Hero, Footer, etc.) para facilitar a manutenção.
  - **Responsividade:** Design adaptável para dispositivos móveis e desktop.
- **Google Fonts:** Integração da fonte "Ubuntu" para uma tipografia moderna.

---

## 🎨 Funcionalidades

- **Navegação Inteligente:** Menu organizado por categorias (Masculino, Feminino, Outlet) e acesso rápido à conta/carrinho.
- **Seção Hero:** Destaque para o produto principal com chamada para ação (CTA).
- **Categorias Dinâmicas:** Cards interativos para navegação por estilos (Casual, Esporte, Moderno, Futurista).
- **Grid de Produtos:** Layout otimizado para exibição de coleções em destaque.
- **Newsletter e Redes Sociais:** Rodapé completo com formulário de captura e links sociais.
- **Menu Mobile:** Implementado via técnica de "checkbox hack" para funcionamento sem necessidade de JavaScript inicial.

## 📁 Estrutura do Projeto

O projeto segue uma estrutura organizada para facilitar a escalabilidade:

```text
ecommerce-syntaxwear/
├── index.html          # Página principal do site
├── css/                # Diretório de estilos
│   ├── base.css        # Estilos globais e reset de elementos
│   ├── reset.css       # Normalização de estilos padrão dos browsers
│   ├── variables.css   # Definição de tokens de design (fontes/cores)
│   └── components/     # Estilos específicos de componentes
│       ├── footer.css
│       ├── header.css
│       ├── hero.css
│       ├── product-category.css
│       └── product-grid.css
├── images/             # Ativos visuais (logos, ícones e produtos)
│   ├── banners/
│   ├── icons/
│   ├── logo/
│   └── products/
└── js/                 # Scripts (reservado para futuras funcionalidades)
```

## 🛠️ Como Visualizar

Para rodar o projeto localmente, basta seguir estes passos:

1. Clone o repositório ou baixe os arquivos.
2. Navegue até a pasta raiz do projeto.
3. Abra o arquivo `index.html` em qualquer navegador web moderno.
   - *Dica: Use a extensão "Live Server" no VS Code para uma melhor experiência de desenvolvimento.*
